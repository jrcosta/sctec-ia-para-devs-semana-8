# Prompt Vigente: Organizacao Serverless

Organize o repositorio para a aplicacao atual do AgendaEdu:

- `frontend/` como app Next.js unica.
- UI em `frontend/src/ui` e pagina principal em `frontend/src/app/page.jsx`.
- Route Handlers em `frontend/src/app/api`.
- Cliente HTTP em `frontend/src/lib/http`.
- Dominio em `frontend/src/lib/agendaedu`.
- Documentacao ativa em `docs/`.
- Documentacao historica em `docs/descontinuado/`.

Garanta que a estrutura deixe claro que nao existe backend Python no fluxo vigente.
