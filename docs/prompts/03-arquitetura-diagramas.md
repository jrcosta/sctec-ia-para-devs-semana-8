# Prompt Vigente: Arquitetura e Diagramas

Crie diagramas Mermaid que representem a aplicacao atual do AgendaEdu:

- Next.js App Router como entrypoint.
- UI React consumindo rotas relativas `/api`.
- Route Handlers serverless em `frontend/src/app/api`.
- dominio em `frontend/src/lib/agendaedu`.
- cliente HTTP em `frontend/src/lib/http`.
- store em memoria no runtime.

Inclua pelo menos:

- um diagrama de componentes;
- um diagrama de fluxo do cadastro e da listagem;
- um diagrama do plano de prioridades.
