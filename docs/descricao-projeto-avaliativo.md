# Descricao do Projeto

AgendaEdu e uma aplicacao web serverless para organizacao de compromissos escolares.

Stack vigente:

- Next.js App Router.
- React Client Components.
- Next.js Route Handlers para backend.
- Markdown e Mermaid para documentacao.
