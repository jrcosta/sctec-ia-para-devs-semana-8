# Prompt Vigente: Refatoracao

Refatore a aplicacao Next.js para o formato vigente do AgendaEdu:

- UI em `frontend/src/ui`;
- cliente HTTP em `frontend/src/lib/http`;
- Route Handlers em `frontend/src/app/api`;
- dominio em `frontend/src/lib/agendaedu`.

Nao altere o contrato JSON dos endpoints existentes e mantenha o store em memoria somente para o ciclo local.
