# PR develop para main: ciclo 3

## Objetivo

Abrir o Pull Request de `develop` para `main` registrando o ciclo 3 do projeto.

## Descricao do ciclo

Este ciclo representa a etapa de refatoracao e mudanca de stack do AgendaEdu.

## Escopo descrito no PR

- Refatoracao da aplicacao para arquitetura fullstack em Next.js.
- Mudanca de stack para reduzir complexidade operacional.
- Uso de Route Handlers do Next.js para o backend serverless.
- Consolidacao da experiencia principal em uma unica aplicacao.
- Documentacao da reducao de escopo motivada por restricao de tempo.

## Motivo

A mudanca foi adotada para reduzir o escopo tecnico da entrega, diminuir a quantidade de partes moveis e facilitar demonstracao, manutencao e validacao dentro do tempo disponivel.
