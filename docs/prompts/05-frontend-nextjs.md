# Prompt Vigente: Fullstack Next.js

Gere a interface atual do AgendaEdu em Next.js.

A experiencia deve incluir:

- formulario para cadastro de compromisso;
- filtros de listagem;
- lista de compromissos;
- visualizacao do plano de prioridades;
- consumo de rotas relativas `/api`;
- tratamento de estado vazio, carregando e erro.

Use a pagina principal em `frontend/src/app/page.jsx`, orquestracao em `frontend/src/App.jsx` e componentes de interface em `frontend/src/ui`.
